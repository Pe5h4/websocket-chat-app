# Basic chat app using WebSocket

## Install packages

### Client

`cd webSocketClient`
`yarn install`

### Server

`cd webSocketServer`
`yarn install`

## Start the server

`cd webSocketServer`
`yarn start`

It will create server connection on port `8090`

## Start the client app

`cd webSocketClient`
`yarn start`

It will open a new tab of your browser on port `1992`
